# Design System: TBWPP

## 1. Overview

**Creative North Star: "The Swiss Wedding Zine"**

A high-contrast, typographically-driven landing page that borrows from Swiss poster design but stays warm and human. The system is built on tension: black and white severity cut with a single vibrant accent, tight display type softened by approachable body copy, editorial confidence that never tips into preciousness. This is a wedding brand that looks like it belongs in a design studio, not a bridal boutique.

The design rejects aspirational wedding aesthetics — no soft pastels, no script fonts, no filtered imagery of perfect couples in golden hour. It also rejects corporate wedding-industry polish (The Knot, Zola). Instead it channels the energy of a well-designed independent zine: bold, direct, slightly irreverent, and unmistakably intentional.

**Key Characteristics:**
- High-contrast black and white foundation
- One vibrant accent color used sparingly and deliberately
- Bold, tight display typography (Swiss poster influence)
- Generous whitespace as a structural element, not decoration
- Warm and approachable despite the typographic severity
- Feels like a cool younger brand, never precious or traditional

## 3. Typography

**Display Font:** [font pairing to be chosen at implementation — direction: Helvetica Neue or similar grotesque sans, bold weight, tight letter-spacing]
**Body Font:** [same family, regular weight, standard tracking for readability]

**Character:** Swiss poster meets podcast personality. Display type is dense, commanding, and unapologetic — tight tracking, heavy weight, large scale. Body type opens up: standard spacing, comfortable reading weight, generous line height. The contrast between the two registers creates the system's warmth within its severity.

### Hierarchy
- **Display** (Bold/Black, [large clamp], tight line-height, negative letter-spacing): Hero headlines, section titles. The Swiss poster voice.
- **Headline** (Bold, [medium], tight line-height): Sub-sections, feature callouts.
- **Title** (SemiBold, [small-medium]): Card titles, testimonial attribution.
- **Body** (Regular, [16-18px], 1.6 line-height, 65-75ch max width): Long-form copy, descriptions, testimonials.
- **Label** (Medium, [small], slight letter-spacing, uppercase): Tags, metadata, navigation items, CTAs.

### Named Rules
**The Tight-Then-Open Rule.** Display type is packed tight (negative tracking, minimal leading). Body type breathes (standard tracking, generous leading). Never reverse this — loose headlines and tight body copy destroys the system's rhythm.

## 4. Elevation

Flat by default. No drop shadows, no layered cards, no depth illusions. The system conveys hierarchy through scale, weight, whitespace, and the single accent color. If something needs to stand out, make it bigger or bolder — don't float it.

### Named Rules
**The No Shadow Rule.** Shadows are prohibited. Depth comes from typography scale and whitespace, not elevation. If a section needs separation, use generous spacing or a full-bleed background color shift (black to white, white to black).

## 6. Do's and Don'ts

### Do:
- **Do** use extreme type scale contrast — massive display headlines next to quiet body copy.
- **Do** let whitespace do structural work. Empty space is not wasted space.
- **Do** use the accent color only on elements that demand action (signup form, primary CTA).
- **Do** keep imagery real and diverse — the brand celebrates all couples, all budgets.
- **Do** write copy in Michelle's voice — direct, warm, slightly irreverent. "Perfection is a myth" energy.
- **Do** use full-bleed black/white section inversions for visual rhythm.

### Don't:
- **Don't** use pastels, gradients, or soft color palettes — this is not an aspirational wedding brand.
- **Don't** use script fonts, handwriting fonts, or serif display fonts — nothing that signals "traditional wedding."
- **Don't** use filtered, golden-hour, Pinterest-style wedding photography.
- **Don't** mimic The Knot, Zola, or WeddingWire's corporate wedding-industry aesthetic.
- **Don't** add shadows, glows, or glassmorphism. The system is flat.
- **Don't** use gradient text, decorative border-left stripes, or hero-metric templates.
- **Don't** make it feel precious, delicate, or exclusive — this brand is accessible, real, and a little bit loud.
- **Don't** chase trends (boho, cottagecore, maximalist florals) — Swiss fundamentals don't age.
